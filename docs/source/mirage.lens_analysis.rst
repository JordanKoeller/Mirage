mirage.lens\_analysis package
=============================

|lens_analysis| is the main entry point for |Mirage|'s analysis tools and interactive environment. Provides many convenience functions and objects for organizing, displaying, and analyzing simulation results. Also includes tools for setting up simulations to be ran.

.. note:: Throughout |Mirage|'s documentation, we assume you alias `lens_analysis` as `la`. That is, you import `lens_analysis` via the command

        >>> import mirage.lens_analysis as la
